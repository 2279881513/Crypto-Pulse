## 📋 功能特性

| 功能 | 说明 |
|------|------|
| **趋势结构** | 道氏理论：基于摆动高低点（pivot）识别趋势方向 |
| **支撑阻力** | 自动识别关键支撑/阻力位，突破/回踩入场 |
| **成交量确认** | 所有信号必须有成交量放大（>MA20×1.5）才执行，且K线收盘在极端位 |
| **止盈止损** | 止盈 = 入场价 ± max(波幅, ATR×2.0)，止损 = ATR × 1.5，追踪止损保本 |
| **止损冷却** | 止损触发后暂停交易5分钟，冷却期内不开新仓 |
| **保本检查** | TP1利润 ≥ 0.10%（双边手续费）才进场，不达标则放弃并标记💰 |
| **风控标记** | 被风控阻止的信号：利润不足→💰，其他→⛔ 橙色箭头标记 |
| **风控统计** | 底部面板显示风控原因分布，可点击高亮对应K线 |
| **仓位管理** | 本金、杠杆、费率可调，自动计算净利、ROI、手续费 |
| **保本过滤** | 严格0.10%阈值过滤低质量交易，利润不足自动放弃 |
| **连续亏损保护** | 同方向连续亏损3次后暂停该方向交易 |
| **交易明细** | 点击某条记录自动跳转到对应K线并闪烁标记 |
| **数据导出** | CSV导出，文件名精确到秒 |
| **日期历史** | 输入框自动保存历史查询记录（localStorage） |

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install flask pandas numpy loguru aiohttp websocket-client requests
```

### 2. 配置 API

编辑 `.env` 文件，填入你的 OKX API 密钥：

```
OKX_API_KEY="你的 API Key"
OKX_SECRET_KEY="你的 Secret Key"
OKX_PASSPHRASE="你的Passphrase"
```

### 3. 下载数据

```bash
python scripts/update_data.py --intervals 1m,5m,15m
```

### 4. 启动系统

```bash
python run_web.py
```

浏览器打开 `http://127.0.0.1:8080`

## 🖥️ 界面说明

### 回测页面（首页 `/`）

- **时间选择**：文本输入框（支持 `2026/05/01 00:00 - 2026/05/31 00:00` 格式）或日期时间选择器
- **日期历史**：点击输入框显示历史查询记录，自动保存到 localStorage
- **快捷按钮**：1天/3天/7天/30天/90天
- **到最新**：自动刷新到最新数据
- **实时模式**：加载7天历史预热，从现在开始模拟交易
- **验证值**：入场后检查多少根K线（5/10/20/30）
- **保本开关**：过滤利润不够扣手续费的交易

### 底部面板

| 标签 | 内容 |
|------|------|
| 📋 交易明细 | 每笔交易的入场/出场/PnL |
| 📊 PnL分布 | 累计盈亏曲线 |
| 💰 盈亏 | 盈亏分布统计 |
| 📊 评分 | 评分分布 + 离场原因分布（含超时盈利/亏损细分） |
| 📶 信号明细 | 总信号数、做多/做空占比 |
| ⚠️ 风控 | 风控阻止统计 + 风控原因分布 + 规则说明 |

### K线图交互

- **鼠标悬停**：显示K线详情、信号评分、交易入场/出场信息
- **紫色出场线**：悬停在交易K线上显示出场价格线（✨止盈/🛑止损/⏱️超时）
- **橙色箭头** ⛔：被风控阻止的信号
- **橙色柱条**：止损冷却区间（5根K线）
- **右键菜单**：放大/缩小/重置/预测K线/导出CSV
- **高亮功能**：点击评分/离场原因/风控原因，K线图上只显示匹配信号

## ⚙️ 回测参数

### 止盈止损

| 参数 | 值 | 说明 |
|------|-----|------|
| 止盈倍数 | max(波幅, ATR×2.0) | TP距离 = max(波幅, ATR×2.0)，确保TP/SL≥1.33 |
| 止损倍数 | ATR × 1.5 | 固定ATR倍数 |
| 验证值 | 5~30（可调） | 默认10根K线 |

### 风控规则

| 规则 | 说明 |
|------|------|
| 止损冷却 | 止损触发后暂停交易N分钟（可开关可调），期间不开新仓 |
| 保本检查 | TP1利润 ≥ 0.10% 才进场，不达标则放弃并标记💰 |
| 连续亏损保护 | 同方向连续亏损3次后暂停该方向交易 |
| 追踪止损 | 到达TP1后将止损移至入场价保本 |

### 出场判断

| 结果 | 条件 |
|------|------|
| ✅ 止盈 | K线最高/低价触及止盈价 |
| ❌ 止损 | K线最高/低价触及止损价 |
| ⏱️ 超时 | 验证期内未触及以上两者，用收盘价判断盈亏 |
