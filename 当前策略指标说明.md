# CryptoPulse 回测策略指标说明
> 最后更新: 2026-07-13 | 版本: v2.1

> 基于**价格行为/结构分析**（道氏理论趋势结构 + 支撑阻力突破 + 成交量确认 + 追踪止损）的 BTC 永续合约量化策略

---

## 一、策略核心思想

**不做预测，只做跟随**。完全基于价格行为：

| 要素 | 说明 |
|------|------|
| **趋势结构** | 道氏理论：更高的高点和低点 = 上升，更低的高点和低点 = 下降 |
| **支撑阻力** | 用 pivot 自动识别关键价位 |
| **成交量确认** | 放量 > 1.5×MA20，且 K 线收盘在极端位 |
| **挂单突破** | 入场价 = pivot 价位（模拟 stop 单），不等收盘确认 |
| **追踪止损** | 价格到达 TP1 后止损移至成本价 |

---

## 二、入场信号

### Pivot 检测
- **LOOKBACK** = 30 根K线 | **left=3, right=3**
- pivot high = 左右各3根范围内最高，pivot low = 左右各3根范围内最低

### 入场条件（按优先级）

| 优先级 | 信号 | 做多 | 做空 | 信心 |
|:------:|------|------|------|:----:|
| 1 | **放量破位** | 上升趋势 + 收盘破 pivot high + 多方K线 + 放量 | 下降趋势 + 收盘破 pivot low + 空方K线 + 放量 | 70~75 |
| 2 | **回踩确认** | 低点触支撑但收盘反弹（阳线） | 高点触阻力但收盘回落（阴线） | 65 |
| 3 | **连续实体** | 连续3根阳线 + 放量 + 多方K线 | 连续3根阴线 + 放量 + 空方K线 | 60 |

---

## 三、止盈止损

### 止损
```
ATR × 1.5（固定）
做多：入场价 − 1.5 × ATR
做空：入场价 + 1.5 × ATR
```

### 止盈（v2 改进）
```
TP距离 = max(波幅, ATR × 2.0)    ← 确保 TP/SL ≥ 1.33
TP1 = 入场价 ± TP距离 × 1.0
TP2 = 入场价 ± TP距离 × 1.5
TP3 = 入场价 ± TP距离 × 2.0
```

### 追踪止损（v2 新增）
到达 trail_activate（≈ TP1）后，止损移至入场价保本。

## 四、风控（用户可控）

| 功能 | UI | 说明 |
|------|----|------|
| **止损冷却** | 顶栏「冷却」复选框 + 分钟输入 | 止损触发后暂停 N 分钟交易（默认5分钟） |
| **保本检查** | 顶栏「保本」复选框 | TP1利润不够双边手续费则放弃 |
| **连续亏损保护** | 后端硬编码（3次） | 同方向连续亏损3次暂停该方向 |
