# CryptoPulse 回测策略指标说明
> 最后更新: 2026-07-12

> 基于多技术指标加权评分卡 + 周期共振 + ADX 趋势过滤的 BTC 永续合约量化策略

---

## 一、技术指标总览

| 指标 | 参数 | 用途 |
|------|------|------|
| **EMA** | 5 / 13 / 21 | 趋势排列：多头排列（短 > 中 > 长），空头排列（短 < 中 < 长） |
| **MACD** | 快线5 慢线13 信号线5 | 金叉/死叉 + 柱线伸长方向 |
| **RSI** | 周期14 | 超买 ≥ 70（看空），超卖 ≤ 30（看多） |
| **布林带（BB）** | 周期20, 标准差2.5 | 价格触碰上轨（超买）或下轨（超卖） |
| **ATR** | 周期14 | 波动率，用于止盈止损距离计算 |
| **ADX** | 周期14 | 趋势强度：≥25 有趋势，≥35 强趋势，<25 盘整 |
| **OBV** | 14周期EMA差 | 资金流向验证 |
| **VWAP** | 20周期 | 价格偏离度 |
| **成交量比** | 20周期均量 | 放量/缩量确认 |
| **微观结构** | OBI / 主动成交比 / 大单 | 辅助验证（权重仅5%） |

---

## 二、做多（LONG）— 入场条件

### 硬性准入条件（三者必须同时满足）

| # | 条件 | 说明 |
|---|------|------|
| 1 | **RSI ≤ 30** | 超卖区 |
| 2 | **价格 ≤ 布林下轨 + (中轨 − 下轨) × 0.3** | 价格靠近下轨 |
| 3 | **ADX ≥ 10** | 至少存在微弱趋势 |

### 加权评分卡（总分 > 阈值 且方向为 BULLISH → 做多）

| 评分项 | 权重 | 条件 |
|--------|:----:|------|
| EMA 排列 | 25% | 5 > 13 > 21 多头排列 |
| MACD | 10% | 零上金叉，柱线伸长 |
| RSI | 10% | 超卖区域值 |
| 动量 | 15% | 向上 |
| 布林带 | 10% | 触碰下轨 |
| OBV | 12% | 向上 |
| VWAP | 5% | 价格在 VWAP 上方 |
| 成交量比 | 8% | 放量 |
| 微观结构 | 5% | 正评分 |

> **ADX ≥ 35** → 总分 × 1.15（放大信号），且评分阈值从 ±30 降至 **±15**
>
> **ADX < 25** → 需总分绝对值 ≥ 35 才能入场（过滤盘整期假信号）
>
> **ADX ≥ 50 + 布林下轨触边** → 绕过评分阈值，**直接做多**（强趋势豁免）
>
> **ADX ≥ 50 + 布林上轨触边** → 绕过评分阈值，**直接做空**（强趋势豁免）

---

## 三、做空（SHORT）— 入场条件

### 硬性准入条件（三者必须同时满足）

| # | 条件 | 说明 |
|---|------|------|
| 1 | **RSI ≥ 70** | 超买区 |
| 2 | **价格 ≥ 布林上轨 − (上轨 − 中轨) × 0.3** | 价格靠近上轨 |
| 3 | **ADX ≥ 10，且 (ADX ≤ 45 或 ADX > 50)** | ADX∈(45,50] 时不做空（历史胜率仅33%），但ADX>50+BB上轨时强趋势豁免做空 |

### 加权评分卡（总分 < 阈值 且方向为 BEARISH → 做空）

与做多使用同一套 9 项指标，评分取负值，对称判断：

| 评分项 | 权重 | 条件 |
|--------|:----:|------|
| EMA 排列 | 25% | 5 < 13 < 21 空头排列 |
| MACD | 10% | 零下死叉，柱线缩短 |
| RSI | 10% | 超买区域值 |
| 动量 | 15% | 向下 |
| 布林带 | 10% | 触碰上轨 |
| OBV | 12% | 向下 |
| VWAP | 5% | 价格在 VWAP 下方 |
| 成交量比 | 8% | 放量 |
| 微观结构 | 5% | 负评分 |

---

## 四、观望（NEUTRAL / HOLD）— 不进场条件

| # | 条件 | 说明 |
|---|------|------|
| 1 | **评分绝对值 ≤ 阈值** | 正常阈值30，强趋势(ADX≥35)时阈值降至15 |
| 2 | **ADX < 25 且总分绝对值 < 35** | 盘整期过滤，强制中性 |
| 3 | **置信度 < 35** | 信号置信度不足 |
| 4 | **5m 趋势与 1m 信号冲突** | 周期共振不一致（周期冲突保护） |
| 5 | **TP1 涨幅不足以覆盖手续费** | 预期利润 < 交易成本（保本检查） |
| 6 | **进场冷却期** | 同方向连续信号间隔 < 2根K线时跳过；ADX>50+BB触边时豁免（间隔≥1根K线可进场） |
| 7 | **强趋势方向保护** | ADX>45 且方向与EMA趋势相反时强制观望（不做逆势） |
| 8 | **极端趋势保护** | ADX>80 强制观望，极端单边行情不开新仓 |
| 9 | **止损冷却** | 止损触发后暂停交易5分钟，期间不开新仓 |

---

## 五、止盈（Take-Profit）— 出场规则

### 回测出场规则

| 方向 | 止盈条件 | 止损条件 | 超时处理 |
|:---:|---------|---------|---------|
| 做多 | 盘中最高价 ≥ 入场价 + n × ATR（n=3.0）→ 止盈 ✅ | 盘中最低价 ≤ 入场价 − 2.0 × ATR → 止损 ❌ | 验证期内未触及以上两者，按收盘价判定盈亏 ⏱️ |
| 做空 | 盘中最低价 ≤ 入场价 − n × ATR（n=3.0）→ 止盈 ✅ | 盘中最高价 ≥ 入场价 + 2.0 × ATR → 止损 ❌ | 同上 |

### 参数

| 参数 | 值 | 说明 |
|------|-----|------|
| **risk_n** | 3.0 | 止盈 = 入场价 ± n × ATR（风险系数，通常2~3） |
| **sl_mult** | 2.0 | 止损 = 入场价 ± 2.0 × ATR |
| **lookahead** | 5~30（可调） | 入场后检查多少根K线 |

---

## 六、止损（Stop-Loss）— 出场规则

| 级别 | 止损距离 | 多头止损价 | 空头止损价 |
|:----:|:--------:|-----------|-----------|
| API 回测默认 | ATR × 2.0 | 入场价 − ATR × 2.0 | 入场价 + ATR × 2.0 |

### 回测出场判断

| 方向 | 止损条件 |
|:---:|---------|
| 做多 | 盘中最低价 ≤ 止损价 → 止损出场 |
| 做空 | 盘中最高价 ≥ 止损价 → 止损出场 |

---

## 七、风控系统

### 止损冷却

- 止损触发后，暂停交易 **5 分钟**
- 冷却期内所有新开仓信号被阻止（橙色箭头标记 ⛔）
- 原有持仓平仓、止盈单正常执行
- K线图上以橙色柱条标记冷却区间

### 保本检查

- 进场前计算 TP1 利润百分比
- 若 TP1 利润 ≤ 双边手续费（默认 0.10%）→ 放弃该交易
- 被阻止的信号以橙色箭头标记 ⛔

### 风控显示

| 元素 | 说明 |
|------|------|
| 🟠 橙色箭头 ⛔ | 被风控阻止的信号（做多↑，做空↓） |
| 🟠 橙色柱条 | 止损冷却区间（成交量位置，5根K线） |
| ⚠️ 风控标签页 | 风控阻止统计、原因分布（可点击高亮） |
| 悬浮窗提示 | 悬停在风控阻止的K线上显示风控原因 |

## 八、策略执行流程

```
每根K线
    │
    ├─→ 计算技术指标：EMA(5,13,21) + MACD(5,13,5) + RSI(14) + BB(20,2.5) + ATR(14) + ADX(14) + OBV + VWAP + 成交量比
    │
    ├─→ 硬性条件判断
    │       ├─ 多头: RSI≤30 AND 价格≤BB下轨附近 AND ADX≥10
    │       └─ 空头: RSI≥70 AND 价格≥BB上轨附近 AND ADX≥10 AND (ADX≤45 OR ADX>50)
    │
    ├─→ 强趋势豁免检查 (ADX≥50 + |bb_pos|≥2.0)
    │       ├─ 是 → 绕过评分，直接做多/做空（评分×1.3）
    │       └─ 否 → 继续以下流程
    │
    ├─→ 加权评分卡（9项指标 × 权重） → 总分 -100 ~ +100
    │
    ├─→ 趋势强度修正
    │       ├─ ADX < 25 且总分绝对值 < 35 → 过滤为中性（观望）
    │       └─ ADX ≥ 35 → 总分 × 1.15
    │
    ├─→ 最终决策（动态阈值）
    │       ├─ ADX ≥ 35: 阈值降至 ±15
    │       ├─ 否则: 阈值 = ±30
    │       ├─ 总分 > 阈值  → 做多（LONG）
    │       ├─ 总分 < −阈值 → 做空（SHORT）
    │       └─ 否则 → 观望（NEUTRAL）
    │
    ├─→ 方向保护过滤
    │       ├─ ADX>45 且与EMA趋势相反 → 过滤（不做逆势）
    │       ├─ ADX>80 → 强制观望（极端趋势不开仓）
    │       └─ 通过 → 继续
    │
    ├─→ 冷却期检查
    │       ├─ 同方向间隔<2根K线 → 冷却跳过
    │       ├─ 但ADX>50+BB触边 → 豁免（间隔≥1根可进场）
    │       └─ 通过 → 继续
    │
    ├─→ 止损冷却检查
    │       ├─ 止损触发5分钟内 → 风控阻止（橙色箭头⛔）
    │       └─ 通过 → 继续
    │
    ├─→ 保本检查
    │       ├─ TP1利润 ≤ 双边手续费 → 风控阻止（橙色箭头⛔）
    │       └─ 通过 → 执行交易
    │
    └─→ 计算出场位
            ├─ 止损: ATR × 2.0
            └─ 止盈: ATR × n（n=3.0）
```

## 九、策略参数速查

| 参数 | 值 | 说明 |
|------|-----|------|
| risk_n | 3.0 | 止盈风险系数，止盈 = 入场价 ± n × ATR |
| sl_mult | 2.0 | 止损倍数，止损 = 入场价 ± sl_mult × ATR |
| cool_bars | 2 | 同方向交易冷却（根K线） |
| sl_cooldown | 300s | 止损后冷却时间（5分钟） |
| 默认费率 | 0.05% | 单边手续费 |
| 保本阈值 | 0.10% | 双边手续费（TP1需超过此值才进场） |
