# CryptoPulse 回测策略指标说明
> 最后更新: 2026-07-13 | 版本: v3.0

> 基于**价格行为/结构分析**（道氏理论趋势结构 + 支撑阻力突破 + 成交量确认 + 追踪止损）的 BTC 永续合约量化策略

---

## 一、策略核心思想

**不做预测，只做跟随**。完全基于价格行为：

| 要素 | 说明 |
|------|------|
| **趋势结构** | 道氏理论：更高的高点和低点 = 上升，更低的高点和低点 = 下降 |
| **支撑阻力** | 用 pivot 自动识别关键价位 |
| **成交量确认** | 放量 > 1.5×MA20，且 K 线收盘在极端位 |
| **挂单突破** | 入场价 = pivot 价位（模拟 stop 单），不等收盘确认 |
| **追踪止损** | 价格到达 TP1 后止损移至成本价 |

---

## 二、入场信号

### Pivot 检测
- **LOOKBACK** = 30 根K线 | **left=3, right=3**
- pivot high = 左右各3根范围内最高，pivot low = 左右各3根范围内最低

### 入场条件

| 信号 | 做多 | 做空 | 信心 |
|------|------|------|:----:|
| **放量突破/破位** | 上升趋势 + 收盘破 pivot high + 多方K线 + 放量>1.5× | 下降趋势 + 收盘破 pivot low + 空方K线 + 放量>1.5× | 70~75 |
| **连续实体** | 连续3根阳线 + 放量 + 多方K线 | 连续3根阴线 + 放量 + 空方K线 | 60 |

### K线位置过滤
- 多方破位：收盘 > 开盘 且 实体 > 影线
- 空方破位：收盘 < 开盘 且 实体 > 影线

---

## 三、止盈止损

### 止损
```
ATR × 1.5（固定）
做多：入场价 − 1.5 × ATR
做空：入场价 + 1.5 × ATR
```

### 止盈
```
TP距离 = max(波幅, ATR × 2.0)     ← 确保 TP/SL ≥ 1.33
TP1 = 入场价 ± TP距离 × 1.0
TP2 = 入场价 ± TP距离 × 1.5
TP3 = 入场价 ± TP距离 × 2.0
```

### 追踪止损
到达 trail_activate（≈ TP1）后，止损移至入场价保本。

---

## 四、风控

| 功能 | 说明 |
|------|------|
| **保本检查（严格）** | TP1利润 ≥ 0.10%（双边手续费）才进场，不足则放弃 |
| **止损冷却** | 止损触发后暂停 N 分钟交易（默认5分钟，可开关可调） |
| **连续亏损保护** | 同方向连续亏损3次暂停该方向 |

---

## 五、策略执行流程

```
每根K线 → 取30根K线识别pivot（left=3）
    → 趋势结构判断（道氏理论）
    → 入场信号检查
        ├─ 放量破位（趋势+突破+放量+K线位置）
        └─ 连续实体（3根同向+放量）
    → 止损冷却检查 → 连续亏损检查 → 保本检查（TP1≥0.10%）
    → 执行交易（止损ATR×1.5，止盈max(波幅,ATR×2.0)）
    → 到达TP1后追踪止损保本
```
